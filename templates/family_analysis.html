Cluster Sequences
? Given your input of the maximum number of clusters, the optimal number of clusters will be calculated. It will generate a .CSV file with the sequences, their counts, and their assigned cluster.


Clusters Finder
? This will output the most abundant cluster peak sequences (a larger input number will take longer to run).


Cluster Comparer
? This will output how similar the top clusters are to each other, creating a fitness landscape of relative distances between clusters. This is the same idea as the Fitness landscape network representation figure in PNAS (Chen 2013).

{% if num_rounds > 1 %}

Cluster Tracker
? Generate figures that calculate the fraction abundance of each of the most common clusters in each round of selection. Use this to understand when enrichment occured and how much it varies by round.

{% endif %}
